## 📋 概述

本文档提供了 Ming Status CLI 项目中模块文档的标准模板，确保所有模块文档的一致性和完整性。

## 🎯 模块文档结构

每个模块应包含以下7个标准文档：

## 📋 使用说明

### 1. 创建新模块文档

1. 复制模板文件到模块目录
2. 替换所有 `[占位符]` 为实际内容
3. 根据模块特点调整结构
4. 填写具体内容

### 2. 模板定制

- 根据模块类型调整模板结构
- 添加模块特有的章节
- 删除不适用的部分
- 保持核心结构不变

### 3. 质量检查

使用 [文档检查清单](documentation-standards.md#检查清单) 确保文档质量。
